# Спецификация языка MiniCompiler

## 1. Ключевые слова
- `fn` - объявление функции
- `int` - целочисленный тип
- `float` - тип с плавающей точкой
- `if` - условный оператор
- `else` - альтернативная ветка
- `while` - цикл
- `return` - возврат значения
- `true` - логическая истина
- `false` - логическая ложь

## 2. Идентификаторы
- Начинаются с буквы (a-z, A-Z)
- Могут содержать буквы, цифры и подчеркивание
- Длина до 255 символов
- Регистрозависимые

## 3. Литералы
- **Целые числа**: 0, 42, 1000 (диапазон -2³¹ до 2³¹-1)
- **Числа с точкой**: 3.14, 0.5, 10.0

## 4. Операторы
- Арифметические: `+ - * /`
- Сравнения: `== != < >`
- Присваивание: `=`

## 5. Разделители
- `( )` - круглые скобки
- `{ }` - фигурные скобки
- `;` - точка с запятой
- `,` - запятая

## 6. Комментарии
- Однострочные: `// текст комментария`
